## ssh-remote-assist 简介

该技能用于指导助手**通过本机终端连接远程服务器**，执行诊断与修复操作。适用于：

- 宝塔/BT 面板无法访问
- 服务异常、端口不通
- 需要查看日志/配置并做最小修改
- 需要 SSH 登录排查线上问题

## 使用方式（OpenClaw）

1) 将 skill 放到 OpenClaw 的 `skills/` 目录（本仓库已完成）。
2) 在对话中描述需求，如：
   - “远程连我的服务器帮我修宝塔面板”
   - “SSH 登录排查面板/网站打不开”
3) 助手会按流程收集 SSH 信息并执行排查与修复。

## 常见流程（技能内流程概述）

- 确认故障范围与授权
- 收集 SSH 连接信息（IP/端口/账号/密码或私钥）
- 从本机终端 SSH 登录
- 本机/远程连通性验证
- 读取日志与配置（先读后改）
- 最小化修改 + 重启相关服务
- 回报结果与后续建议

## 安全提示

- 默认**不执行破坏性操作**，除非你明确授权。
- 修改配置前会建议备份。
- 重启服务会提前说明。
